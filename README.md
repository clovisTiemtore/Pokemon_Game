# Jeu de Cartes Pokemon

Ce projet en C++ est une mise en œuvre du système de jeu de cartes Pokémon, illustrant les principes de la Programmation Orientée Objet (POO), tels que l'héritage et la gestion des interactions entre classes.

---

## Fonctionnalités

* **Gestion des cartes** : Implémentation de différents types de cartes (Pokémon, Énergie, Dresseur).
* **Mécaniques de jeu** : Gestion des points de vie (HP), des attaques et des coûts en énergie.
* **Système de joueur** : Gestion du deck et de la main d'un joueur.

---

## Installation et Compilation

**Compilation :**
Utilisez la commande suivante pour compiler l'ensemble des fichiers source :
```bash
g++ main.cpp Card.cpp PokemonCard.cpp EnergyCard.cpp TrainerCard.cpp Player.cpp -o pokemon_game
```
**Execution :**
Sur Windows :
```
.\pokemon_game.exe
```
Sur Linux : 
```
./pokemon_game.exe
```
